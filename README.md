# FitPulse Gym Website

FitPulse is a modern responsive gym website designed as a frontend portfolio project.  
It includes multiple pages, blog articles, trainer showcase, and interactive filtering.

## 🚀 Features

- Responsive design (Desktop / Tablet / Mobile)
- Blog page with category filtering (JavaScript)
- Article pages for fitness tips
- Trainer showcase section
- Membership pricing section
- Contact page with map
- Modern dark UI with orange accent theme

## 🛠 Technologies Used

- HTML5
- CSS3
- JavaScript
- Flexbox & Grid
- Responsive Design (Media Queries)
